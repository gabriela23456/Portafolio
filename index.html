Sobre mí

Soy desarrolladora web con experiencia en aplicaciones full-stack y sistemas de gestión.

Me especializo en construir sistemas que resuelven problemas reales: desde gestión de citas y administración hasta generación automatizada de documentos. Uso tecnologías como PHP, Astro, TypeScript, HTML, CSS y bases de datos SQL para entregar productos mantenibles y escalables.

Busco sumarme a un equipo donde pueda seguir creciendo y aportar con código de calidad y soluciones bien pensadas.
